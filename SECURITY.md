# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Do not open a public issue.

Instead, contact the maintainer directly:

- Email: [mensimohamedamine25@gmail.com](mailto:mensimohamedamine25@gmail.com)
- Or open a private security advisory (if enabled)

Please include:

- A clear description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (if available)

---

## Response Process

Once a vulnerability is reported:

- You will receive an acknowledgment within a few days
- The issue will be investigated and validated
- A fix will be developed and released as soon as possible
- You will be notified when the issue is resolved

---

## Disclosure Policy

- Please do not publicly disclose the vulnerability until it has been fixed
- After resolution, responsible disclosure is allowed

---

## Security Updates

Security updates will be released as needed and documented in the changelog.

---

Thank you for helping keep this project secure.
