# Contributing to Sirocco

Thank you for considering contributing to this project. Contributions of all kinds are welcome, including bug reports, feature requests, and code improvements.

## How to Contribute

### 1. Fork the Repository

Click the **Fork** button and clone your fork locally:

```bash
git clone https://github.com/mensi-commits/sirocco
cd sirocco
```

### 2. Create a Branch

Use a descriptive branch name:

```bash
git checkout -b feature/add-login
```

### 3. Make Your Changes

- Write clear and maintainable code
- Follow the existing project structure and style
- Add comments where necessary

### 4. Commit Your Changes

Use meaningful commit messages:

```bash
git commit -m "feat: add user login functionality"
```

### 5. Push and Open a Pull Request

```bash
git push origin feature/add-login
```

Then open a Pull Request with a clear description of your changes.

---

## Reporting Bugs

When opening an issue, please include:

- A clear description of the issue
- Steps to reproduce
- Expected and actual behavior
- Screenshots if applicable

---

## Suggesting Features

To suggest a feature, include:

- The problem it solves
- Why it would be useful
- Possible alternatives

---

## Code Guidelines

- Keep functions small and focused
- Use descriptive variable names
- Follow the existing formatting
- Add tests if applicable

---

## Pull Request Guidelines

- Keep pull requests focused and concise
- Reference related issues (e.g., `Closes #12`)
- Ensure the code runs correctly
- Update documentation when necessary

---

## Code of Conduct

By contributing, you agree to follow the project's Code of Conduct.

---

Thank you for your contribution.
